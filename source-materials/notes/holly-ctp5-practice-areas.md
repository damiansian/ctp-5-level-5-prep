## Skill Clusters

### High Priority

#### 1. Probability and complements

- Section: Mathematics
- Score: 3/7 (43%)
- Missed questions: Math Q5, Q8, Q31, Q40
- Error pattern: Confuses complement (1 minus probability) with the probability itself. Does not multiply probabilities for independent events.
- Practice focus: `P(A and B) = P(A) x P(B)` for independent events. `P(not A) = 1 - P(A)`. Converting between fractions, decimals, and percentages for probability.

#### 2. Unit conversion and measurement

- Section: Mathematics
- Score: 3/6 (50%)
- Missed questions: Math Q7, Q37, Q38, Q48
- Error pattern: Uses wrong operation (multiply vs. divide) or wrong direction when converting units. Struggles with compound unit conversions (yards/feet/inches).
- Practice focus: Volume = base area x height (and the inverse). Pounds to ounces (x16). Grams to milligrams (x1000). Converting compound measurements to a single unit before subtracting.

#### 3. Exponents and notation

- Section: Mathematics
- Score: 0/2 (0%)
- Missed questions: Math Q9, Q39
- Error pattern: Interprets `8²` as `8 x 2` instead of `8 x 8`. Thinks multiplying by 1000 moves decimal left instead of right.
- Practice focus: `n² = n x n`, `n³ = n x n x n`. Multiplying by powers of 10 moves decimal right. Dividing by powers of 10 moves decimal left.

#### 4. Multi-step word problems

- Section: Quantitative Reasoning
- Score: 2/7 (29%)
- Missed questions: Quant Q22, Q23, Q28, Q31
- Error pattern: Loses track of intermediate steps. Does not work backwards systematically. Confuses "three times a number" with "a number times three" in equation setup.
- Practice focus: Translating English to algebra (`3x - 2 = 58`). Working backwards through chained operations. Age problems requiring multiple time references. Midpoint and halfway calculations for time.

#### 5. Modular/cyclic patterns

- Section: Quantitative Reasoning
- Score: 1/5 (20%)
- Missed questions: Quant Q25, Q30; Verbal Q47
- Error pattern: Off-by-one errors when finding position in repeating sequences. Does not use modular arithmetic (remainder after division).
- Practice focus: For a repeating pattern of length `n`, the `k`th item is at position `(k mod n)`. If remainder is 0, it maps to the last item in the cycle, not the first.

### Medium Priority

#### 6. Subject-verb agreement

- Section: Writing Mechanics
- Score: 0/3 (0%) on this specific skill
- Missed questions: WM Q46, Q58, Q59
- Error pattern: Consistently chose "No error" when the subject and verb did not agree. Does not identify the true subject when prepositional phrases intervene (for example, "The problems with the presentation was" where the subject is "problems," not "presentation").
- Practice focus: Identify the subject and ignore prepositional phrases between subject and verb. Plural subjects take plural verbs. "`There is/are`" depends on what follows.

#### 7. Capitalization rules

- Section: Writing Mechanics
- Score: 11/15 (73%)
- Missed questions: WM Q16, Q18, Q29
- Error pattern: Does not capitalize titles used before names. Misses lowercase proper names. Does not recognize that job titles used as common nouns should not be capitalized.
- Practice focus: Capitalize titles before names. Always capitalize proper names. Do not capitalize job titles used generically.

#### 8. Spelling recognition

- Section: Writing Mechanics
- Score: 12/15 (80%)
- Missed questions: WM Q31, Q35, Q41
- Error pattern: Chose "no error" when misspelled words were present. Missed words like "lonely," "bicycle," and "alligator."
- Practice focus: Common misspelling patterns, particularly vowel placement and double letters.

#### 9. Usage errors

- Section: Writing Mechanics
- Score: 10/15 (67%)
- Missed questions: WM Q48, Q54
- Error pattern: Does not catch verb tense inconsistencies. Misses awkward phrasing.
- Practice focus: Past time markers require past tense verbs. "`To + verb`" versus "`that + verb`" constructions.

#### 10. Punctuation

- Section: Writing Mechanics
- Score: 6/8 (75%)
- Missed questions: WM Q5, Q7
- Error pattern: Missed comma between city and state. Did not catch singular possessive used where plural possessive was needed.
- Practice focus: City and state format always uses a comma. Singular possessive versus plural possessive.

#### 11. Fraction arithmetic

- Section: Mathematics
- Score: Needs practice
- Missed questions: Math Q21, Q42
- Error pattern: Errors adding mixed numbers with different denominators. Did not complete the multi-step sushi roll fraction problem.
- Practice focus: Finding common denominators. Adding and subtracting mixed numbers. Computing fractional parts of a whole number.

#### 12. Reading graphs and data

- Section: Mathematics
- Score: Missed Q40
- Missed questions: Math Q40
- Error pattern: Misread bar graph proportions and computed wrong totals. Did not correctly find total from a known part and its percentage.
- Practice focus: If part = percentage x total, then total = part / percentage. Combining multiple percentages before multiplying.

#### 13. Custom operators

- Section: Quantitative Reasoning
- Score: 0/1
- Missed questions: Quant Q1
- Error pattern: Did not correctly apply the defined operator formula `a ☆ b = (a+b)ab`.
- Practice focus: Substitute values carefully into defined formulas. Compute step by step, not in head.

#### 14. Scientific notation

- Section: Quantitative Reasoning
- Score: 0/1
- Missed questions: Quant Q9
- Error pattern: Miscounted zeros after expanding scientific notation (`2.36 x 10⁴` vs. `4.6 x 10³`).
- Practice focus: Expand scientific notation to standard form, then count zeros. The number of zeros is not the same as the exponent when the coefficient has decimal places.

#### 15. Insufficient information recognition

- Section: Quantitative Reasoning
- Score: 1/2
- Missed questions: Quant Q10, Q14
- Error pattern: Chose a definite answer when the problem did not provide enough information to determine the relationship.
- Practice focus: If a problem has unknown variables and no way to pin them down, the answer is D. Check whether you can construct more than one valid outcome.

#### 16. Vocabulary in context

- Section: Vocabulary
- Score: 16/20 (80%)
- Missed questions: Vocab Q46, Q48, Q54, Q57, Q60
- Error pattern: Second-guessing correct instincts. Initially selects a correct answer then changes to a wrong one. Struggles with similar-sounding words.
- Practice focus: Read the full sentence for context clues before looking at options. Trust first instinct if it fits the sentence logic. Learn the differences between similar-sounding words.

### Low Priority

#### 17. Logic/elimination puzzles

- Section: Verbal Reasoning
- Score: 9/15 (60%)
- Missed questions: VR Q53, Q54, Q55, Q59
- Error pattern: Does not systematically work through each constraint before selecting an answer. Misses process-of-elimination steps in multi-person or multi-attribute problems.
- Practice focus: Write down each constraint. Eliminate options one at a time. Check that the final answer satisfies all constraints.

#### 18. Analogies

- Section: Verbal Reasoning
- Score: 22/25 (88%)
- Missed questions: VR Q7, Q18, Q21
- Error pattern: Misidentifies the relationship type, particularly for opposite or antonym pairs and "trap vs. release" style relationships.
- Practice focus: Name the relationship before looking at answers.

#### 19. Classification headings

- Section: Verbal Reasoning
- Score: 18/20 (90%)
- Missed questions: VR Q26, Q42
- Error pattern: Occasionally selects a member of the category instead of the heading word. Confused "Body" with "Essay" in one case.
- Practice focus: The heading is the broadest word that includes all others as examples or parts.

#### 20. Passage-based vocabulary

- Section: Vocabulary
- Score: 18/20 (90%)
- Missed questions: Vocab Q21, Q24
- Error pattern: Wavered between a synonym and the actual best answer. Second-guessed correct choices.
- Practice focus: Substitute each option into the original sentence. The one that preserves the meaning most closely is correct.
